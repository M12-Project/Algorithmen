# 📘 Sorting Algorithms Visualizer

An interactive web application built with Vite, React, and Node.js that visually demonstrates how different sorting algorithms work. The tool animates each step of the algorithm, allowing users to adjust array size, animation speed, and algorithm type in real time.

---

## 🧩 Overview

The Sorting Algorithms Visualizer helps users understand how common sorting algorithms operate internally. Each algorithm is displayed as a bar chart that updates dynamically as the algorithm progresses. The project is ideal for students, developers, and anyone who wants to explore algorithmic behavior visually.

---

## ⭐ Features

- Real‑time visualization of sorting algorithms  
- Adjustable array size  
- Adjustable animation speed  
- Multiple algorithms (e.g., Bubble Sort, Merge Sort, Quick Sort, Insertion Sort)  
- Clean and responsive UI  
- Built with Vite for fast development and optimized builds
